# مخيم الملفى - نسخة HTML & CSS

تم تحويل التطبيق من React/Vite إلى HTML و CSS خالص.

## الملفات

- `index.html` - الصفحة الرئيسية
- `booking.html` - صفحة الحجز
- `styles.css` - ملف الأنماط الرئيسي
- `script.js` - ملف JavaScript للتفاعلات

## كيفية الاستخدام

1. افتح `index.html` في المتصفح مباشرة
2. أو استخدم خادم محلي:
   ```bash
   # باستخدام Python
   python -m http.server 8000
   
   # باستخدام Node.js (http-server)
   npx http-server
   ```

## الروابط

- الصفحة الرئيسية: `index.html`
- صفحة الحجز: `booking.html`

## التخصيص

### تغيير رقم الواتساب
ابحث عن `971501234567` في الملفات واستبدله برقمك

### تغيير إحداثيات الموقع
ابحث عن `25.0657,55.1713` واستبدلها بإحداثيات موقعك

### تغيير روابط التواصل الاجتماعي
ابحث عن `REPLACE_INSTAGRAM_USERNAME` و `REPLACE_TIKTOK_USERNAME` واستبدلها

### تغيير صورة Hero
استبدل `src/assets/hero-bg.jpg` بصورتك الخاصة
